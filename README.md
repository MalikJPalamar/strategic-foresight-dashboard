# Strategic Foresight Dashboard

**Live:** [centaurion.me/strategic-foresight-dashboard](https://centaurion.me/strategic-foresight-dashboard)

A consolidated strategic intelligence dashboard applying the **FTI (Future Today Institute) 7-Step Forecasting Methodology** to three businesses — Alchemy of Breath (AOB/ASHA), BuilderBee, and Centaurion.me — against the backdrop of the 2026 Iran conflict (Operation Epic Fury, Day 40) and the validated Gulf Expat Exodus opportunity.
